# Визуальные идеи для «Ставь Угольки»

## 1. Ember premium
Тёмный фон как уголь, мягкие красно-оранжевые свечения на активных элементах, кремовый текст и холодный бирюзовый акцент от логотипа.

## 2. Живой баннерный верх
Баннеры без лишних надписей: огонь, искры, дым, фактура угля. Один сильный визуал = один слайд.

## 3. Товарные карточки как витрина
Большая фотография, округлённые углы, две маленькие иконки поверх фото, цена и корзина в одной линии. Без лишних подпоясняющих текстов.

## 4. Нижнее меню как glass-bar
Снизу плавающая панель с мягким затемнением, чтобы выглядело как настоящее мобильное приложение, а не сайт.

## 5. Микроанимации
Лёгкий подъём карточки при тапе, мягкое перелистывание баннеров, пульсирующий бейдж корзины.

## 6. Владелец как control room
Для owner-интерфейса лучше другая логика: чистые таблицы, графичные карточки аналитики, спокойный интерфейс без перегруза декоративными эффектами.
