# Ставь угольки — Telegram Mini App магазин

Готовый проект магазина для Telegram Mini App в стиле вашего логотипа.

## Что внутри

- **Витрина для клиентов** — мобильная, адаптирована под открытие внутри Telegram.
- **Панель владельца** — управление товарами, заказами, статусами и аналитикой.
- **Файловое хранилище JSON** — без сложной базы данных, можно быстро запустить и доработать.
- **Опциональный Telegram-бот** — кнопка открытия магазина и прием `web_app_data` из Mini App.

## Структура

- `public/index.html` — клиентская витрина.
- `public/admin.html` — страница владельца.
- `server.js` — API, авторизация админа, товары, заказы, аналитика.
- `bot.js` — Telegram-бот для открытия магазина и получения заказов.
- `data/*.json` — товары, заказы, настройки.

## Быстрый запуск

1. Установите Node.js 20+.
2. Скопируйте `.env.example` в `.env`.
3. Запустите:

```bash
npm install
npm run start
```

После запуска:

- магазин: `http://localhost:3000/`
- админка: `http://localhost:3000/admin.html`

Демо-логин для админки берется из `.env`:

- `ADMIN_USERNAME=owner`
- `ADMIN_PASSWORD=stavugolki2026`

## Переменные окружения

```env
PORT=3000
PUBLIC_BASE_URL=http://localhost:3000
WEB_APP_URL=http://localhost:3000
ADMIN_USERNAME=owner
ADMIN_PASSWORD=stavugolki2026
JWT_SECRET=please-change-me
BOT_TOKEN=
ADMIN_CHAT_ID=
```

## Telegram Mini App

Чтобы открыть магазин из Telegram:

1. Создайте бота через BotFather.
2. Укажите `WEB_APP_URL` в `.env`.
3. Запустите бота:

```bash
npm run bot
```

4. В BotFather настройте кнопку меню / web app на ваш публичный URL.
5. Если заполнить `ADMIN_CHAT_ID`, бот будет пересылать данные заказов владельцу.

## Что умеет владелец

- добавлять, редактировать и удалять товары;
- переключать наличие и хит продаж;
- менять статусы заказов;
- смотреть выручку, средний чек, последние заказы и топ-товары;
- редактировать текст магазина и контакты.

## Деплой

### Через Docker

```bash
docker compose up --build
```

### Без Docker

Подойдет любой Node-хостинг: VPS, Render, Railway, Fly.io, Timeweb, Beget и т.д.

## Важно перед реальным запуском

- поменяйте пароль админа;
- поменяйте `JWT_SECRET`;
- замените демо-товары на реальные;
- укажите ваш Telegram username в настройках магазина;
- поставьте публичный HTTPS-домен для Mini App.
