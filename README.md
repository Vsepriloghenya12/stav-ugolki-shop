# Ставь угольки — Telegram Mini App

Готовый проект магазина для Telegram Mini App:
- мобильная клиентская витрина;
- нижнее меню;
- карточки товаров 2 в ряд;
- баннеры-карусель;
- поиск;
- избранное;
- корзина и оформление заказа;
- кабинет владельца;
- товары, баннеры, заказы, аналитика, настройки.

## Запуск

```bash
cp .env.example .env
npm install
npm start
```

## Адреса

- магазин: `http://localhost:3000/`
- владелец: `http://localhost:3000/admin.html`

## Демо вход владельца

- логин: `owner`

## Что редактируется владельцем

- товары;
- баннеры;
- статусы заказов;
- базовые настройки магазина;
- аналитика по выручке и категориям.

## Telegram бот

Есть `bot.js`.

Для подключения заполните `.env`:
- `BOT_TOKEN`
- `WEB_APP_URL`
- `ADMIN_CHAT_ID` — необязательно
