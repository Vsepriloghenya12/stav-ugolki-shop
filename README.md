# Ставь угольки — no-deps build

Эта версия не требует `express`, `dotenv`, `jsonwebtoken` или других npm-зависимостей для запуска сервера.

## Запуск

```bash
npm start
```

или

```bash
node server.js
```

## Адреса

- магазин: `/`
- владелец: `/admin.html`

## Данные владельца

- логин: `owner`
- пароль: `stavugolki2026`

## Переменные окружения

- `PORT`
- `ADMIN_USERNAME`
- `ADMIN_PASSWORD`
- `JWT_SECRET`
